📋 테스트용 구글 시트 생성 가이드

1단계: 구글 시트 생성

아래 버튼을 클릭하여 새로운 구글 시트를 생성하세요:

새 구글 시트 만들기

2단계: 시트 제목 설정

시트 이름을 다음과 같이 변경하세요:

  • 2025-01-22 배송관리 (오늘 날짜로 변경)
  • 또는 테스트 배송시트

3단계: 배송 데이터 입력

다음과 같은 형식으로 데이터를 입력하세요:

A열: 고객명 B열: 연락처 C열: 주소 D열: 배송상태
홍길동 010-1234-5678 서울시 강남구 테헤란로 123 대기
김철수 010-9876-5432 부산시 해운대구 마린시티 456 준비중
이영희 010-5555-7777 대구시 중구 동성로 789 출발
박민수 010-3333-9999 인천시 연수구 송도국제도시 101 완료

4단계: 시트별 배송자 지정

배송자별로 시트를 나누려면:

  • 하단의 "시트1" 탭을 우클릭
  • "이름 바꾸기" 선택
  • 배송자 이름으로 변경 (예: "홍길동", "김철수")
  • 시트 추가 버튼(+)으로 배송자별 시트 추가

5단계: 스프레드시트 ID 복사

브라우저 주소창에서 스프레드시트 ID를 복사하세요:

https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgvE2upms/edit#gid=0

복사할 부분: /d/ 뒤부터 /edit 앞까지의 긴 문자열

6단계: 공유 설정

API 접근을 위해 시트를 공유 설정하세요:

  1. 우측 상단 "공유" 버튼 클릭
  2. "링크를 아는 모든 사용자가 편집 가능" 설정
  3. 또는 구글 서비스 계정 이메일 추가

⚠️ 주의사항

  • 실제 운영 시에는 보안을 위해 서비스 계정을 사용하세요
  • 테스트용으로만 "링크를 아는 모든 사용자" 설정을 사용하세요
  • 개인정보가 포함된 실제 데이터는 사용하지 마세요

✅ 준비 완료!
